# Transitia Milano — sito vetrina (Next.js)

Questo pacchetto contiene una landing completa (Milano) + sezioni: Come funziona, Sicurezza, FAQ, CTA.
Stile: colori rilassanti + foto generiche (Unsplash).

## Requisiti
- Node.js 20+ (consigliato)
- Git (opzionale ma utile)

## Avvio in locale
```bash
npm install
npm run dev
```
Apri: http://localhost:3000

## Pubblicazione gratis (consigliato): Vercel
1. Crea un account su Vercel
2. Carica questo progetto su GitHub (o usa import diretto)
3. Su Vercel: **Add New Project** → importa repo → Deploy

## Alternative gratis
- Netlify
- Cloudflare Pages

## Note
- Le immagini sono link esterni (Unsplash). Se vuoi 100% offline, puoi scaricarle e metterle in /public.
